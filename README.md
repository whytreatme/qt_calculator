# QT Calculator - 带括号运算的图形化计算器

一个基于 **C++** 与 **QT框架** 实现的带图形界面的计算器。本项目不仅核心采用了**状态机设计模式**来管理基础逻辑，更实现了**表达式解析算法**，能够正确处理带括号的复杂运算优先级，展现了扎实的软件架构与算法设计能力。

## 🚀 功能特点

- **基础运算**：支持加 (+)、减 (-)、乘 (*)、除 (/) 四则运算。
- **高级功能**：**完整支持括号运算**，可处理复杂表达式如 `(1 + 2) * (3 - 4)`。
- **连续计算**：支持连续运算，上一次的结果可作为下一次计算的初始值。
- **交互设计**：提供清空 (C)、退格 (⌫) 功能，实时显示输入表达式。

## 🌟 设计亮点

### 1. 状态机架构
- 采用**状态机设计模式**，将计算器的交互逻辑抽象为清晰的状态（如`输入左操作数`、`输入运算符`、`输入右操作数`、`显示结果`）。
- 有效替代了复杂的`if-else`条件分支，使代码结构清晰、易于维护和扩展。

### 2. 表达式解析算法
- 为实现**括号运算功能**，实现了完整的表达式解析与求值模块。
- 该模块能够**正确识别并处理操作符优先级和括号的嵌套关系**，确保了复杂表达式计算的准确性。
- 这是本项目区别于简单演示项目的核心技术，体现了扎实的算法功底。

## 🚦 构建与运行

### 环境要求
- QT 5.x 或更高版本
- C++11 兼容的编译器

### 编译步骤
1. 克隆本仓库：
   ```bash
   git clone https://github.com/<你的用户名>/qt-calculator.git
   cd qt-calculator    
2.使用 QT Creator 打开 calculator.pro 文件。    
3.配置构建套件后，点击"构建"并"运行"。
